# 望海镇 — 治愈系游戏设计文档

> 一款关于"自愈与愈人"的温暖治愈游戏。玩家来到海边小镇望海镇，在与居民的深度连接中，进入他们的心象空间，同时面对自己的内心。没有杀戮，但有冲突；没有标准答案，但有成长。

---

## 一、核心情感弧线：交织模型

**自愈 ⇄ 愈人，同步缠绕，互相触发。**

- 向内看：面对自己的创伤、恐惧、阴影
- 向外伸出手：遇见需要帮助的人，尝试给予温暖。有时成功，有时搞砸
- 被反弹回来：帮助被拒绝、好心办坏事——这些时刻反过来逼人审视自己

四种关键戏剧场景：
- **拒绝型**："你根本不懂我的感受"——对方说的没错
- **越界型**：急于"修好"别人，其实是为了逃避自己的无力感
- **镜像型**：对他人的苛责，其实是对自己的不接纳
- **沉默型**：什么都不做，只是陪在旁边——"那天你在，就够了"

设计原则：不设正确选项、失败即成长、拒绝是真实的、回环叙事。

---

## 二、世界观：双层结构

### 表层 · 海边小镇（魔幻现实主义）
安静的海边小镇，有灯塔、老街、码头、一家茶馆。居民过着普通生活，但眼底藏着故事。玩家在这里遇见、倾听、建立信任。

### 里层 · 心象空间（奇幻寓言）
每个人内心的独特世界——内疚是一片暴风雨之海，失语是一个没有面孔的画中世界，治愈者的孤独是一座倒流的茶馆。没有对白，只有隐喻和体验。玩家的行动本身就是疗愈。

**信任是钥匙**：只有建立足够深的连接，才能进入心象空间。

---

## 三、玩家定位：同行者

不是英雄，不是心理咨询师，不是救世主。是一个带着自己的伤来到小镇的人，在帮别人的同时找自己的路。

---

## 四、任务结构：螺旋推进模型

### 主线（主角的疗愈之路）
分阶段解锁，不是线性推进：
1. 抵达小镇，遇见茶馆老板阿树（引路人）
2. 在帮别人时触碰旧伤 → 主角心象空间第一次打开
3. 进入自己的心象空间（核心关卡）
4. 在愈人愈己的循环中达成和解
5. 结局不设"完美"，只有"完整"

### 支线（居民心象之旅）
每位居民一条独立支线，彼此交织。支线完成后主角获得"领悟"，解锁主线新阶段。支线可以不按顺序做，也可以失败。

### 螺旋推进
相遇 → 进入心象 → 在帮助中触发旧伤 → 主角心象解锁新区域 → 获得领悟回到小镇。每次与他人的深度连接，自然揭开内心的一层。

---

## 五、心象空间交互（四种模式）

不同 NPC 搭配不同模式，每个心象有独特手感：

| 模式 | 核心动词 | 体验 |
|------|---------|------|
| 触摸与修复 | 拾起、擦拭、拼合、点亮 | 修好一盏灯，亮起一段记忆 |
| 共鸣与回应 | 聆听、跟随、和声、等待 | 在正确的时刻静止，用声音回应旋律 |
| 选择与见证 | 观察、理解、选择、承受 | 观看记忆剧场，选择如何回应 |
| 陪伴与存在 | 坐在一起、一起走路、什么也不做 | "你在吗？""在。" |

---

## 六、冲突系统（四种形态）

冲突不是为了赢，而是为了守住边界、表达真相、保护重要的人。

| 类型 | 场景 | 玩法 | 成长主题 |
|------|------|------|---------|
| 以武止戈 | 心象中的"内在暴君"、欺凌 | 节奏型防御，气势对抗 | 勇气 |
| 辩论 | NPC的固执信念、观念冲突 | 四种策略（逻辑/共情/提问/袒露） | 智慧 |
| 争吵 | 与亲近之人的关系张力 | 情绪强度条，争吵后的沉默是真正的关卡 | 诚实 |
| 无视/沉默 | 无意义挑衅、语言无法触及的时刻 | 不做任何操作就是选项 | 定力 |

冲突后果不是胜负，而是关系的改变。每一次冲突都是一次人际关系的学习。

---

## 七、暗面系统：人人都有阴影

### 核心原则
风和日丽时人人善良，风暴来临时每个人的贪嗔痴慢疑都会浮出来——包括主角，也包括那些需要帮助的 NPC。人性如水温，不同温度下不同形态。游戏不评判哪种形态是"真的"，而是让玩家看见全部。

### 主角的五毒（贪嗔痴慢疑）
在主角的心象空间中，五毒以五个暗面化身出现。不是要被消灭的BOSS，而是要被看见、承认、接纳的自我碎片。越对抗越强大，越承认越平静。

| 五毒 | 表现 | 接纳之路 |
|------|------|---------|
| 贪 | 想被所有人喜欢，帮别人其实是为了收集感激和证明自己有用 | 承认"我需要被需要"，帮助可以同时为了自己和他人 |
| 嗔 | 对伤害过自己的人感到怒火，有时波及无辜 | 愤怒是边界的守护者，以武止戈的能量来源 |
| 痴 | 逃避真相，用忙碌和帮助他人来逃避看自己的内心 | 停下来，坐在最不想看的角落 |
| 慢 | "我已经走出来了，我比他们强"的优越感 | 你不是老师，只是一个也在迷路的人 |
| 疑 | 不相信自己真的能改变，怀疑他人的善意 | 在不确定中仍然选择伸出手 |

### NPC 的暗面（日常面 vs 触发面）
每个 NPC 在平常都是美好的，但在痛处被触碰时会暴露暗面：

| NPC | 日常面 | 触发面 | 暴露的五毒 |
|-----|--------|--------|-----------|
| 阿树 | 温暖智慧的茶馆老板 | 当你试图帮他时攻击你："你帮得了谁？你连自己都没修好！" | 慢、嗔、疑 |
| 老陈 | 沉默温柔的灯塔守灯人 | 当你安慰"不是你的错"时怒斥你："你根本不知道那晚发生了什么！" | 嗔、疑、痴 |
| 小满 | 安静画画的女孩 | 当你帮她画出母亲的脸时撕画怒吼，然后哭着道歉 | 嗔、贪 |
| 秋姨 | 热情温暖的面馆老板娘 | burnout 下口不择言："你以为人人都跟你一样可以说走就走？"随即愣住道歉 | 嗔、慢、痴 |

### 设计意图
不是"揭露真面目"——日常面和触发面都是真实的。在安全时人的确美好，在恐惧时人的确会攻击。看见全部，才是真正的理解。这才是"灵魂的升华与成长"——不是变成没有暗面的人，而是敢于转身面对自己的阴影。

## 八、情感三幕式节奏

如同一首乐曲，情感需要有呼吸的起伏——不能一路沉重到底，也不能一路甜蜜到底。

### 整体情感曲线
```
第一幕 温暖引入 (暖色60%)  →  第二幕 深入暗面 (冷暖交替)  →  第三幕 整合升华 (平静60%)
──────────────────╮          ╭──────────────────╮          ╭──────────────────
                  ╰──╮    ╭──╯                  ╰──╮    ╭──╯
                     ╰────╯  最暗时刻：五毒对峙     ╰────╯  尾声：灯塔上的平静
```

### 第一幕：抵达 · 温暖如初见
开场仪式（家、行李、猫）→ 抵达望海镇 → 散步、发现小镇之美 → 第一晚的孤独感（小低谷）→ 遇见阿树（回暖）。情绪目标：让玩家放下戒备，爱上这个世界。

### 第二幕：深入 · 一波比一波深
进入老陈心象空间（第一个小高潮）→ 钓鱼喝茶（呼吸）→ 小满撕画争吵（更高浪）→ 海边和解（修复）→ 阿树拒绝帮助（更深裂痕）→ 秋姨的面（喘息）→ 主角心象空间全开，五毒对峙（最暗时刻）。情绪目标：在沉重中让玩家感到"原来我可以是这样的"。

### 第三幕：归来 · 不完美但完整
再次敲阿树的门，喝茶沉默（回暖）→ 阿树心象空间，陪伴而非修复（最终高潮）→ 小镇日常回归，但一切不同了（平缓释放）→ 灯塔顶上的夕阳，猫在身边（尾声）。情绪目标：深深的、有重量的平静。

### 节奏规则
- 波峰后必有波谷：每次沉重之后必须有日常来呼吸
- 最暗处有微光：五毒对峙中也有温暖的记忆碎片
- 最高处有阴影：最温馨的场景里也有一丝忧郁
- 情感分配：第一幕暖60%/忧25%/争5%/静10%，第二幕暖30%/忧30%/争25%/静15%，第三幕暖25%/忧10%/争5%/静60%

## 九、开场仪式：离家之前

**场景**：傍晚的公寓。窗外晚霞。茶几上放着去望海镇的车票。行李箱还能塞三样东西。

**物品与低语**：十几件散落家中的物品，每件点击后会"说话"——幽默、自嘲、温柔的提问。每件物品代表一种情感母题（告别、被爱、未了之事、重量、共鸣、韧性）。

**隐藏彩蛋 — 猫**：一只安静蜷在沙发上的猫。无提示、无高亮。玩家需要主动发现并决定带它走。猫不是物品，不能"塞进行李箱"——带它走意味着打破规则。唯一回馈：它陪着你。

**选择的影响**：带走的物品和猫会在后续游戏中反复出现，成为主角心象空间的主题元素。

---

## 十、实现路径：B + C 并行，三步走

1. **互动小说先行**：用纯文本写出第一章（开场 → 抵达小镇 → 阿树 → 老陈支线），验证故事情感
2. **AI 生成 DEMO 并行**：生成老陈心象空间的可玩 DEMO，验证"触摸与修复"交互
3. **汇合到轻量引擎**：故事文本和核心交互验证通过后，用 Godot 整合成完整游戏

|-----|------|------|---------|---------|
| 阿树 | 茶馆老板，主角引路人 | 治愈了所有人，从未治愈自己 | 倒流的茶馆 | 陪伴与存在 |

## 十二、与传统 RPG 的根本区别

| 维度 | 传统 RPG | 本作 |
|------|---------|------|
| 驱动方式 | 任务日志 + 地图标记 | 情感共鸣 + 好奇心驱动 |
| 任务逻辑 | 接受 → 完成 → 领奖 | 连接 → 进入心象 → 互相改变 |
| 失败处理 | 重来 / 读档 | 搞砸是剧情的一部分，不可逆但有新路 |
| 探索收益 | 经验值 / 装备 / 金币 | 故事碎片 / 领悟 / 新对话选项 / 心象新区域 |
| 战斗 | 消灭对手 | 四种冲突形态，目标是表达立场而非消灭 |
